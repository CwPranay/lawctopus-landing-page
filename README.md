# Lawctopus Course Landing Page

A modern, conversion-focused redesign of the Lawctopus course page for Mastering Contract Drafting & Freelancing. The project preserves the essential course information while delivering a cleaner user experience, stronger visual hierarchy, and a fully responsive presentation.

---

## Live Demo

https://lawctopus-landingpage.vercel.app/

---

## Overview

This project reimagines the original course page into a polished landing experience designed to improve engagement and drive enrollment. The redesign focuses on:

- Better user experience
- Modern UI
- Clear information hierarchy
- Conversion-focused layout
- Fully responsive design

---

## Features

- Modern responsive landing page
- Sticky navigation with smooth scrolling
- Conversion-focused Hero section
- Course highlights
- Learning outcomes
- Career support section
- Faculty showcase
- Pricing section
- Money-back guarantee
- FAQ accordion
- Final CTA
- Responsive footer
- Smooth animations
- Accessible components
- Optimized images
- Mobile-first design

## Installation

```bash
git clone https://github.com/CwPranay/lawctopus-landing-page.git
cd lawctopus-landing-page
npm install
npm run dev
```

---

## Build

```bash
npm run build
npm run start
```
